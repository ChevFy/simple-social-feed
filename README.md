**Prerequisites**
- **.NET SDK:** 9.0 or newer
- **Editor:** VS Code (recommended)
- **OS:** Linux/macOS/Windows

**Setup**
- Open this folder in VS Code.
- Restore dependencies:

```bash
dotnet restore
```

**Run**
- Build and start:

```bash
dotnet build
dotnet run
```

- App will listen on:
  - http: http://localhost:5264
